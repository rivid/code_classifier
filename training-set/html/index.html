人脸关键点检测

ReadFace人脸关键点检测可以精确定位面部的主要关键点，包括眉毛，眼睛，鼻子，嘴巴等。

表情单元检测

ReadFace 人脸AU检测可以对面部44个表情单元进行精确定位并追踪，覆盖人类3000多种表情变化。

情绪识别

ReadFace情绪识别技术可以自动识别照片、视频中人脸表达的情绪，包括喜悦、悲伤、恐惧、厌恶、惊讶、轻蔑、中性等。

认知状态识别

ReadFace 人脸认知状态识别技术可以自动识别照片、视频中人脸的认知状态，包括疲劳、思考、认真观看、走神等。

面部动作识别

ReadFace 面部动作识别主要包括头部姿态识别，如点头、摇头、左右转头；和面部动作识别，如张嘴、合嘴、吐舌头、睁眼、闭眼等。
